# Predicting-Hospital-Readmission-of-Diabetic-Patient
Hospital readmissions are a major challenge in healthcare systems. Patients who are readmitted within 30 days of discharge often indicate issues in treatment, discharge planning, or follow-up care.

This project builds a machine learning model to predict hospital readmission risk for diabetic patients using historical hospital encounter data.

The model helps healthcare providers identify high-risk patients and intervene early, potentially reducing healthcare costs and improving patient outcomes.
